> **Before you can read the map, you must know the name of the land.**

Virtauma means current. Not the static kind. The kind that moves through a channel
it has carved for itself over time, that knows where it is going because it has
been going there long enough to remember.

This is a documentation and content platform built with Nuxt. It has a theme, a
database, an auth layer, a search, a cache strategy, a changelog, a roadmap. Each
of these has a name from Finnish folklore. Not for decoration. Because names carry
meaning, and meaning makes systems easier to think about.

## Miten lukea - How to read

The guides are written for people who are continuing, not arriving. If something
assumes you know what came before, that is intentional. Read Aino if you want to
know what has happened. Read Ajatar if you want to know what is coming. Read
Pellervoinen if you want to know what needs doing.

The language is Finnish and English together. Not translated, not side by side.
The same thought in both, because some things sit better in one than the other.

## Mistä aloittaa - Where to start

If you are new, start with Manala. It tells you how the two worlds connect and
where everything lives. Then Vipunen, then Tiera. The rest will follow.

If you are continuing someone else's work, start with Pellervoinen. Then Aino.
Then come back here.

**The current knows the way. Follow it.**
