## colors Värit 
Virtauma tarjoaa Tailwind CSS-käyttöluokkia jokaiselle määrittelemällesi [semanttiselle värille](/fi/docs/getting-started/theme/design-system), jonka ansiosta voit käyttää luokkanimiä, kuten `text-error` tai `bg-success`:

::code-preview
[Ensisijainen]{class="text-primary text-sm px-4 py-1.5 inline-block"}
[Toissijainen]{class="text-secondary text-sm px-4 py-1.5 inline-block"}
[Onnistuminen]{class="text-success text-sm px-4 py-1.5 inline-block"}
[Tietoa]{class="text-info text-sm px-4 py-1.5 inline-block"}
[Varoitus]{class="text-warning text-sm px-4 py-1.5 inline-block"}
[Virhe]{class="text-error text-sm px-4 py-1.5 inline-block"}

#code
```vue
<template>
  <span class="text-primary">Ensisijainen</span>
  <span class="text-secondary">Toissijainen</span>
  <span class="text-success">Onnistuminen</span>
  <span class="text-info">Tietoa</span>
  <span class="text-warning">Varoitus</span>
  <span class="text-error">Virhe</span>
</template>
```
::

Jokainen käyttöluokka sisältää CSS-muuttujia, jotka sisältävät tumman ja vaalean mallin:

::code-group
```css [Vaalea]
:root {
  --ui-primary: var(--ui-color-primary-500);
  --ui-secondary: var(--ui-color-secondary-500);
  --ui-success: var(--ui-color-success-500);
  --ui-info: var(--ui-color-info-500);
  --ui-warning: var(--ui-color-warning-500);
  --ui-error: var(--ui-color-error-500);
}
```

```css [Tumma]
.dark {
  --ui-primary: var(--ui-color-primary-400);
  --ui-secondary: var(--ui-color-secondary-400);
  --ui-success: var(--ui-color-success-400);
  --ui-info: var(--ui-color-info-400);
  --ui-warning: var(--ui-color-warning-400);
  --ui-error: var(--ui-color-error-400);
}
```
::

::tip
Voit määritellä, mitä sävyjä mikäkin käyttöluokka sisältää tummassa ja vaaleassa mallissa `main.css` tiedostossa:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-primary: var(--ui-color-primary-700);
}

.dark {
  --ui-primary: var(--ui-color-primary-200);
}
```
::

::warning
Et voi käyttää  `primary: 'black'` [**config**](/fi/docs/getting-started/theme/design-system):ssa, koska `black` ei sisällä useita vivahteita.  Käyttääksesi täysin tummaa tai valkoista ensisijaisena värinä, aseta se suoraan `main.css`-tiedostoon:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-primary: black;
}

.dark {
  --ui-primary: white;
}
```

::

## Teksti

Virtauma tarjoaa Tailwind CSS-käyttöluokkia tekstin väriin, jolloin voit käyttää luokkanimiä, kuten `text-dimmed` tai `text-muted`:

::code-preview
[Himmeä]{class="text-dimmed text-sm px-4 py-1.5 inline-block rounded-md"}
[Mutetettu]{class="text-muted text-sm px-4 py-1.5 inline-block rounded-md"}
[Sävytetty]{class="text-toned text-sm px-4 py-1.5 inline-block rounded-md"}
[Teksti]{class="text-default text-sm px-4 py-1.5 inline-block rounded-md"}
[Korostettu]{class="text-highlighted text-sm px-4 py-1.5 inline-block rounded-md"}
[Käänteinen]{class="text-inverted bg-inverted text-sm px-4 py-1.5 inline-block rounded-md"}

#code
```vue
<template>
  <span class="text-dimmed">Himmeä</span>
  <span class="text-muted">Mutetettu</span>
  <span class="text-toned">Sävytetty</span>
  <span class="text-default">Teksti</span>
  <span class="text-highlighted">Korostettu</span>
  <span class="text-inverted bg-inverted">Käänteinen</span>
</template>
```
::
Jokainen käyttöluokka sisältää CSS-muuttujan, jotka asettaa värit tummassa ja vaaleassa mallissa:

::code-group

```css [Vaalea]
:root {
  --ui-text-dimmed: var(--ui-color-neutral-400);
  --ui-text-muted: var(--ui-color-neutral-500);
  --ui-text-toned: var(--ui-color-neutral-600);
  --ui-text: var(--ui-color-neutral-700);
  --ui-text-highlighted: var(--ui-color-neutral-900);
  --ui-text-inverted: var(--color-white);
}
```

```css [Tumma]
.dark {
  --ui-text-dimmed: var(--ui-color-neutral-500);
  --ui-text-muted: var(--ui-color-neutral-400);
  --ui-text-toned: var(--ui-color-neutral-300);
  --ui-text: var(--ui-color-neutral-200);
  --ui-text-highlighted: var(--color-white);
  --ui-text-inverted: var(--ui-color-neutral-900);
}
```

::

::tip
Voit määrittää CSS-muuttujat `main.css`-tiedostossa:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-text: var(--ui-color-neutral-900);
}

.dark {
  --ui-text: var(--color-white);
}
```
::

## Radius
Virtauma ohittaa Tailwind CSS:n `rounded-*`-oletusasetukset omalla reunasäde-järjestelmällä, joten voit käyttää yleisiä [reunan säteen määrityksiä](https://tailwindcss.com/docs/border-radius), kuten `rounded-xs` tai `rounded-2xl`:

::code-preview
[xs]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-xs mr-2"}
[sm]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-sm mr-2"}
[md]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-md mr-2"}
[lg]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-lg mr-2"}
[xl]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-xl mr-2"}
[2xl]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-2xl mr-2"}
[3xl]{class="border-2 border-accented text-sm px-4 py-1.5 inline-block rounded-3xl mr-2"}

#code
```vue
<template>
  <div class="rounded-xs">xs</div>
  <div class="rounded-sm">sm</div>
  <div class="rounded-md">md</div>
  <div class="rounded-lg">lg</div>
  <div class="rounded-xl">xl</div>
  <div class="rounded-2xl">2xl</div>
  <div class="rounded-3xl">3xl</div>
</template>
```
::

Nämä käyttöluokat perustuvat yleisen `--ui-radius` CSS-muuttujan laskemiseen, joka määrittää perussäteen arvon kaikissa komponenteissa yhtenäisen ulkoasun saavuttamiseksi.

```css
:root {
  --ui-radius: 0.25rem;
}
```

::tip
Voit määritellä näitä CSS-muuttujia `main.css`-tiedostossa:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-radius: 0.5rem;
}
```
::

::note
Kokeile :prose-icon{name="i-lucide-swatch-book" class="text-primary"} teeman valitsinta otsikkopalkissa muuttaaksesi perussädettä.
::

## Kontti

Virtauma tarjoaa `--ui-container` CSS-muuttujan, joka määrittää [Container](https://ui.nuxt.com/docs/components/container)-konttikomponentin maksimileveyden.

```css
:root {
  --ui-container: var(--container-7xl);
}
```

::tip
Voit määritellä näitä CSS-muuttujia `main.css`-tiedostossa säätääksesi konttien leveydet yhtenäisesti:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme {
  --container-8xl: 90rem;
}

:root {
  --ui-container: var(--container-8xl);
}
```
::

## Otsikkopalkki

Virtauma tarjoaa `--ui-header-height` CSS-muuttujan, joka määrittää [Header](https://ui.nuxt.com/docs/components/header)-otsikkokomponentin korkeuden.

```css
:root {
  --ui-header-height: --spacing(16);
}
```

::tip
Voit määritellä näitä CSS-muuttujia `main.css`-tiedostossa säätääksesi otsikkopalkin korkeuden yhtenäisesti:

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

:root {
  --ui-header-height: --spacing(24);
}
```
::

## Body

Virtauma määrittää `<body>`{lang="html"}-elementin oletusluokat yhtenäisen teeman tummassa ja vaaleassa mallissa:

```css
body {
  @apply antialiased text-default bg-default scheme-light dark:scheme-dark;
}
```
