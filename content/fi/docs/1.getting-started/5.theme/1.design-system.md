## Tailwind CSS

Tailwind CSS:sä käytetään CSS ensin-kokoonpanoa, jossa voit määritellä suunnittelumerkkejä käyttäen [`@theme`](https://tailwindcss.com/docs/functions-and-directives#theme-directive)-määritettä suoraan CSS:ssä. Tämän ansiosta teema on siirrettävä, ylläpidettävä ja helposti muokattavissa.

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme {
  /* Omat suunnitelmamerkit tähän */
}
```

::note{to="https://tailwindcss.com/docs/theme" target="_blank"}
Tutustu Tailwind CSS-dokumentaaatiossa kaikkiin mahdollisiin muuttujien vaihtoehtoihin.
::

### Kirjaisimet

Käytä `--font-*` teeman muuttujia [määrittääksesi kirjainperheen ominaisuudet](https://tailwindcss.com/docs/font-family#customizing-your-theme) projektissasi.

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme {
 --font-sans: "Finlandica", "sans-serif";
  --font-display: "Arima Madurai", "display";
  --font-mono: "Chivo Mono", "mono";
  --font-hand: "Caveat", "handwriting";
}
```

### Värit

Käytä `--color-*` teeman muuttujia [määtittääksesi värit](https://tailwindcss.com/docs/colors#customizing-your-colors) tai [ohita oletusasetukset](https://tailwindcss.com/docs/colors#overriding-default-colors).

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme static {
  --color-green-50: #EFFDF5;
  --color-green-100: #D9FBE8;
  --color-green-200: #B3F5D1;
  --color-green-300: #75EDAE;
  --color-green-400: #00DC82;
  --color-green-500: #00C16A;
  --color-green-600: #00A155;
  --color-green-700: #007F45;
  --color-green-800: #016538;
  --color-green-900: #0A5331;
  --color-green-950: #052E16;

  /* Uusi väri */
  --color-brand-50: #fef2f2;
  --color-brand-100: #fee2e2;
  --color-brand-200: #fecaca;
  --color-brand-300: #fca5a5;
  --color-brand-400: #f87171;
  --color-brand-500: #ef4444;
  --color-brand-600: #dc2626;
  --color-brand-700: #b91c1c;
  --color-brand-800: #991b1b;
  --color-brand-900: #7f1d1d;
  --color-brand-950: #450a0a;
}
```

::warning
Kun lisäät värejä, varmista että kaikki sävyt `50` ja `950` välillä löytyy.
::

### Keskeytyspisteet

Käytä `--breakpoint-*` teeman muuttujia [määrittääksesi keskeytyspisteet](https://tailwindcss.com/docs/responsive-design#customizing-your-theme).

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme {
  --breakpoint-3xl: 1920px;
  --breakpoint-4xl: 2560px;
  --breakpoint-5xl: 3840px;
}
```

## Värit

Virtauma:n värijärjestelmä perustuu **semanttiseen nimeämiseen** yksittäisten väriarvojen sijaan. Tämän ansiosta ulkoasu on päivitettävissä ja mahdollistaa helpon teemojen vaihdon.

### Semanttiset värit

Virtauma tarjoaa Semanttiset värialiakset, jotka kuvaavat värin **käyttötarkoitusta**. Jokainen alias perustuu väriin `@theme`-asetuksissa, joka voi olla mikä tahansa [Tailwind CSS-oletuspaletista ](https://tailwindcss.com/docs/colors).

| Väri | Oletus | Kuvaus |
| --- | --- | --- |
| `primary`{color="primary"} | `green` | CTA:t, aktiivinavigaatio, brändielementit, tärkeät linkt |
| `secondary`{color="secondary"} | `blue` | Toissijaiset napit, vaihtoehtoiset toiminnot, ulkoasua täydentävät |
| `success`{color="success"} | `green` | Onnistuneet viestit, valmistuneet osiot, positiiviset varmistukset |
| `info`{color="info"} | `blue` | Tietohuomautukset, työkaluvihjeet, aputekstit, neutraalit ilmoitukset |
| `warning`{color="warning"} | `yellow` | Varoitusviestit, keskeneräiset tilat, huomiota vaativat jutut |
| `error`{color="error"} | `red` | Virheviestit, varmennusvirheet, tuhoisat toiminnot |
| `neutral` | `slate` | Teksti, reunat, taustat, käytöstä postetut tilat |

Nämä semanttiset värit ovat saatavilla `color`-ominaisuudella Virtauma-komponentessa.

::note
Kokeile :prose-icon{name="i-lucide-swatch-book" class="text-primary"} teeman valitsinta otsikkopalkissa nähdäksesi miten eri värimaailmat vaikuttavat koko ulkoasuun!
::

### Suorituksenaikainen määritys

Voit muokata värejä suoritusaikana [`app.config.ts`](https://nuxt.com/docs/guide/directory-structure/app-config#app-config-file)-tiedostossa `ui.colors`-avaimen alla, joka mahdollistaa dynaamisen teeman muokkaamisen ilman palvelimen uudelleenkäynnistystä:

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    colors: {
      primary: 'blue',
      secondary: 'purple',
      neutral: 'zinc'
    }
  }
})
```

::caution
Voit käyttää vain teemassa olevia värejä. Joko:
- Käytä [Tailwindin oletusvärejä](https://tailwindcss.com/docs/colors) (kuten `blue`, `green`, `zinc`)
- Määritä omat värit ensin käyttäen `@theme`-määritystä (kuten `brand` ylläolevassa esimerkissä)
::

### Laajennetut värit

Voit haluta määrittää muita semanttisia värejä oletusten lisäksi, kuten `tertiary`:

Ensin rekisteröi uusi väri `nuxt.config.ts`-tiedostoon `ui.theme.colors`-avaimella:

```ts [nuxt.config.ts] {7}
export default defineNuxtConfig({
  ui: {
    theme: {
      colors: [
        'primary',
        'secondary',
        'tertiary',
        'info',
        'success',
        'warning',
        'error'
      ]
    }
  }
})
```

Sitten voit määrittää sen `app.config.ts`-tiedostossa `ui.colors`-avaimella:

```ts [app.config.ts] {6}
export default defineAppConfig({
  ui: {
    colors: {
      primary: 'blue',
      secondary: 'purple',
      tertiary: 'indigo'
    }
  }
})
```
Lopuksi tätä väriä voidaan käyttää komponenteissa, jotka tukevat `color`-ominaisuutta tai [luokkaa](/fi/docs/getting-started/theme/css-variables):

```vue
<UButton color="tertiary">
  Erikoistoiminto
</UButton>
```
