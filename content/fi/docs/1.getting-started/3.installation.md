## `create-nuxt` CLI

### Luo kansio

Käytä `create nuxt@latest` terminaalissa luodaksesi uuden projektin:

```bash [Terminal]
npm create nuxt@latest
```

Suositellaan `bun`:a paketinhallinnassa.

Paikallinen esikatselu löytyy osoitteesta `http://localhost:3000`
