## Yleinen rakenne

Virtauma on **Nuxt-kerros** , joka laajentaa perus Nuxt-ohjelmaa dokumentaatio-ominaisuuksilla. Tämä mahdillistaa täyden Nuxt-tuen.

Kun luot uuden projektin käyttämällä `npm create nuxt@latest`, saadaan:

```
├── content/             # Markdown-sisältö
│   ├── en/index.md      # Kotisivu Englanniksi
│   │    └── docs/       # Dokumentaatio Englanniksi
│   ├── fi/index.md      # Kotisivu Suomeksi
│        └── docs/       # Dokumentaatio Suomeksi
├── public/              # Staattinen sisältö, kuvat
└── package.json         # Ruuppuvuudet ja skriptit
```

Voit käytttää mitä tahansa Nuxtin ominaisuutta tai tiedostoa:

```
├── app.config.ts        # Ohjelman asetukset
├── nuxt.config.ts       # Nuxtin asetukset (lisää moduuleja, komponentteja jne.)
├── app/                 # Ohjelman hakemisto
│   ├── components/      # Komponentit (lisää omia)
│   ├── layouts/         # Asettelut (lisää omia)
│   └── pages/           # Sivut (lisää omia)
└── server/              # Palvelinkoodi (lisää omia)
```

### `content/`-hakemisto
Tänne lisätään Markdown-sivut. Reitit luodaan automaattisesti tiedostorakenteiden avulla.

**Yksikielinen rakenne:**
```
content/
├── index.md
├── getting-started.md
└── guide/
    ├── introduction.md
    └── configuration.md
```

**Monikielinen rakenne (käyttäen i18n):**
```
content/
├── en/
│   ├── index.md
│   └── guide/
│       └── introduction.md
└── fi/
    ├── index.md
    └── guide/
        └── introduction.md
```

::tip
Lisätietoa **i18n**-käännösominaisuuksista löytyy [Käännökset](/fi/docs/getting-started/localization)-osiosta.
::

### `public/`-hakemisto

`public/`- hakemistossa olevat tiedostot palvellaan juurelta, eikä niitä muokata rakennusprosessissa. Täältä löytyy kuvat, kuvakkeet, sekä muut staattiset tiedostot.

### `package.json`

Tässä tieostossa on kaikki ohjelmiston riippuvuudet ja skriptit. 

Virtauma-ohjelmiston `package.json` on todella minimaalinen ja näyttää tältä:

```json [package.json]
{
  "name": "docs",
  "scripts": {
    "build": "nuxt build",
    "dev": "nuxt dev",
  },
  "dependencies": {
    "better-sqlite3": "^12.2.0",
    "nuxt": "^4.0.0"
  }
}
```

### `nuxt.config.ts`

**Tämä tiedosto ei ole pakollinen Virtaumaissa.**

Voit käyttää lisämoduuleja Nuxt-asetustiedostossa:

```typescript [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['@nuxtjs/i18n']
})
```

### `app.config.ts`

**Tämä tiedosto ei ole pakollinen Virtaumaissa.**

Täältä löytyy [Virtauma-asetukset](/fi/docs/getting-started/usage), joita voi muokata sopimaan teemaan, hoitaa SEO:t, sekä lisätä linkkejä.

## Täysi Nuxt-projektipotentiaali

Koska Virtauma on Nuxt-kerros, voit käyttää **mitä tahansa** Nuxt-ominaisuutta:

```
my-docs/
├── app/                 # Ohjelmiston hakemisto (valinnainen)
│   ├── components/      # Omat Vue-komponentit
│   ├── layouts/         # Omat asettelut
│   ├── pages/           # Omat Vue-sivut (ei `/content`)
│   ├── composables/     # Vue sävellykset (koodinpätkät)
│   └── middleware/      # Reittien väliohjelmisto 
├── server/              # Palvelinkoodi
│   └── api/             # API-reitit
├── plugins/             # Nuxt-laajennukset
├── middleware/          # Yleinen väliohjelmisto
└── modules/             # Omat Nuxt-moduulit
```

::tip{to="/fi/docs/getting-started/usage"}
Tämän kerrostyyppisen lähestymistavan myötä voit käyttää koko Nuxt-järjestelmää keskittyen sisällön luomiseen.
::
