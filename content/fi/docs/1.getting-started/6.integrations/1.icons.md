## Käyttö

Virtauma rekisteröi automaattisesti [`@nuxt/icon`](https://github.com/nuxt/icon)-moduulin, joten ylimääräistä säätöä ei tarvita.

### Kuvakekomponentti

Voit käyttää [Icon](https://ui.nuxt.com/docs/components/icon)-komponenttia `name`-ominaisuudella näyttääksesi kuvakkeen:

::code-preview

:::icon
---
name: 'i-lucide-lightbulb'
class: 'size-5'
---
:::

#code

```md
::icon
---
name: 'i-lucide-lightbulb'
class: 'size-5'
---
::
```

::

::note
Voit käyttää mitä tahansa nimeä <https://icones.js.org>-kokoelmasta.
::

### Komponenttiominaisuudet

Joissain komponenteissa on `icon`-ominaisuus kuvakkeen näyttämiseksi, kuten [Button](https://ui.nuxt.com/docs/components/button):

::code-preview

:::button
---
icon: i-lucide-sun
variant: subtle
---
Painike
:::

#code

```md
::button
---
icon: i-lucide-sun
variant: subtle
---
Painike
::
```

::

## Kokoelmat

### Iconify dataset

Suositellaan asentamaan kuvakedata paikallisesti käyttäen:

::code-group{sync="pm"}

```bash [bun]
bun i @iconify-json/{collection_name}
```

```bash [pnpm]
pnpm i @iconify-json/{collection_name}
```

```bash [yarn]
yarn add @iconify-json/{collection_name}
```

```bash [npm]
npm install @iconify-json/{collection_name}
```

::

Esimerkiksi käytettäessä `i-tabler-github`-kuvaketta asenna kokoelma käyttäen `@iconify-json/tabler`. Näin ollen kuvakkeet voidaan tuoda paikallisesti palvelittomien toimintojen avulla, joka on nopeampaa ja luotettavampaa niin SSR:ssä kuin asiakaspuolella.

::note{to="https://github.com/nuxt/icon?tab=readme-ov-file#iconify-dataset" target="_blank"}
Lue aiheesta lisää `@nuxt/icon`-dokumentaatiosta.
::

### Omat paikalliset kokoelmat

Voit käyttää paikallisia SVG-tiedostoja luodaksesi oman Iconify-kokoelman.

Esimerkiksi voit pistää kuvakkeiden SVG-tiedostot valitsemaasi kansioon, kuten `./app/assets/icons`:

```bash
assets/icons
├── add.svg
└── remove.svg
```

`nuxt.config.ts`-tiedostossa lisää kohde `icon.customCollections`:

```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui'],
  css: ['~/assets/css/main.css'],
  icon: {
    customCollections: [{
      prefix: 'custom',
      dir: './app/assets/icons'
    }]
  }
})
```

Sitten voit käyttää kuvakkeita näin:

```vue
<template>
  <UIcon name="i-custom-add" />
</template>
```

::note{to="https://github.com/nuxt/icon?tab=readme-ov-file#custom-local-collections" target="_blank"}
Lue aiheesta lisää `@nuxt/icon`-dokumentaatiosta.
::
