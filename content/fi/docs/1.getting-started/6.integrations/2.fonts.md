## Käyttö

Virtauma rekisteröi [`@nuxt/fonts`](https://github.com/nuxt/fonts)-moduulin automaattisesti, jolloin ylimääräistä säätöä ei tarvita.

### Julistus

Käyttääksesi kirjaisimia Virtauma:ssa voit julistaa ne CSS:ssä. Ne ladataan ja optimoidaan automaattisesti.

```css [app/assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme {
  --font-sans: 'Finlandica', sans-serif;
}
```

### Kokoonpano

Voit poistaa `@nuxt/fonts`-moduulin käytöstä `ui.fonts`-valinnalla `nuxt.config.ts`-tiedostossa:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  ui: {
    fonts: false
  }
})
```
