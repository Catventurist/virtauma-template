## Käyttö

::note
Virtauma tarjoaa **App**-komponentin, joka yhdistää ohjelmiston ja yleisasetukset.
::

### Kieli

Käytä `locale`-ominaisuutta valittuna `@nuxt/ui/locale`:

```vue [app.vue]
<script setup lang="ts">
import { fr } from '@nuxt/ui/locale'
</script>

<template>
  <UApp :locale="fr">
    <NuxtPage />
  </UApp>
</template>
```

### Oma kieli

Voit luoda uuden kielen käyttäen `defineLocale`-koostettavaa:

```vue [app.vue]
<script setup lang="ts">
import type { Messages } from '@nuxt/ui'

const locale = defineLocale<Messages>({
  name: 'My custom locale',
  code: 'en',
  dir: 'ltr',
  messages: {
    // lisää parit
  }
})
</script>

<template>
  <UApp :locale="locale">
    <NuxtPage />
  </UApp>
</template>
```

::tip
Katsoen `code`-parametriä, kielen iso-koodi pitää lisätä. Esim:

* `hi` Hindi (kieli)
* `de-AT`: Saksan (kieli), jota käytetään Itävallassa (alue)

::

### Kielen laajennos

Voit muokata olemassaolevia kieliä ohittaen `messages` tai `code` käyttäen `extendLocale`-koostettavaa:

```vue [app.vue]
<script setup lang="ts">
import { en } from '@nuxt/ui/locale'

const locale = extendLocale(en, {
  code: 'en-GB',
  messages: {
    commandPalette: {
      placeholder: 'Search a component...'
    }
  }
})
</script>

<template>
  <UApp :locale="locale">
    <NuxtPage />
  </UApp>
</template>
```

### Dynaminen kieli

Vaihtaaksesi kieltä dynaamisesti voit käyttää [Nuxt I18n](https://i18n.nuxtjs.org/)-moduulia.

::steps{level="4"}

#### Asenna Nuxt I18n-paketti

::code-group{sync="pm"}

```bash [pnpm]
pnpm add @nuxtjs/i18n
```

```bash [yarn]
yarn add @nuxtjs/i18n
```

```bash [npm]
npm install @nuxtjs/i18n
```

```bash [bun]
bun add @nuxtjs/i18n
```

::

#### Lisää Nuxt I18n-moduuli `nuxt.config.ts`{lang="ts-type"}-tiedostoon

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: [
    '@nuxt/ui',
    '@nuxtjs/i18n'
  ],
  css: ['~/assets/css/main.css'],
  i18n: {
    locales: [{
      code: 'de',
      name: 'Deutsch'
    }, {
      code: 'en',
      name: 'English'
    }, {
      code: 'fr',
      name: 'Français'
    }]
  }
})
```

#### Aseta `locale`-ominaisuus käyttäen `useI18n`

```vue [app.vue]
<script setup lang="ts">
import * as locales from '@nuxt/ui/locale'

const { locale } = useI18n()
</script>

<template>
  <UApp :locale="locales[locale]">
    <NuxtPage />
  </UApp>
</template>
```

::

### Dynaminen suunta

Jokaisessa kieless on `dir`-ominaisuus, jota käytetään `App`-komponentissa asettamaan komponenttien suuntaus.

Monikielisessä sovelluksessa on suositeltavaa käyttää `lang` ja `dir`-ominaisuuksia `<html>`-elementissä dynaamisesti perustuen käyttäjän kieleen, joka hoituu[useHead](https://nuxt.com/docs/api/composables/use-head)-koostettavalla:

```vue [app.vue]
<script setup lang="ts">
import * as locales from '@nuxt/ui/locale'

const { locale } = useI18n()

const lang = computed(() => locales[locale.value].code)
const dir = computed(() => locales[locale.value].dir)

useHead({
  htmlAttrs: {
    lang,
    dir
  }
})
</script>

<template>
  <UApp :locale="locales[locale]">
    <NuxtPage />
  </UApp>
</template>
```
