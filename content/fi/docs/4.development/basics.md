## Verkkokehitys on monimutkaista

- HTML (HyperText Markup Language): Jokaisen verkkosivun perusta. Se määrittelee verkkosisällön rakenteen.
- CSS (Cascading Style Sheets): Käytetään HTML-elementtien tyylittelyyn. Se tekee verkkosivuista visuaalisesti houkuttelevampia ja käyttäjäystävällisempiä.
- JavaScript: Ohjelmointikieli, joka tekee verkkosivuista interaktiivisia. Sitä käytetään dynaamisen verkkosisällön luomiseen.
- HTTP (HyperText Transfer Protocol): Protokolla, jota käytetään tiedon siirtämiseen internetissä. Se on kaiken verkon tiedonvaihdon perusta.
- REST (Representational State Transfer): Arkkitehtuurinen tyyli tietokonejärjestelmien välisten standardien tarjoamiseen verkossa.
- JSON (JavaScript Object Notation): Kevyt tiedonvaihtoformaatti, joka on helppo ihmisten lukea ja kirjoittaa sekä koneiden jäsentää ja tuottaa.
- AJAX (Asynchronous JavaScript and XML): Joukko verkkosovellusteknologioita, jotka käyttävät monia verkkoteknologioita asiakaspuolella asynkronisten verkkosovellusten luomiseen.
- DOM (Document Object Model): Ohjelmointirajapinta verkkodokumenteille. Se esittää dokumentin rakenteellisena solmujen ja objektien ryhmänä, joilla on erilaisia ominaisuuksia ja metodeja.
- API (Application Programming Interface): Joukko sääntöjä, jotka määrittävät, kuinka eri ohjelmistosovellukset kommunikoivat keskenään.
- Versionhallintajärjestelmät (VCS): Työkaluja, kuten Git, käytetään koodin eri versioiden hallintaan.
- Responsiivinen verkkosuunnittelu: Verkkosivujen tekeminen hyväksi kaikilla laitteilla (pöytätietokoneet, tabletit ja puhelimet).
- Saavutettavuus: Varmistetaan, että verkkosivustot ovat kaikkien käytettävissä, myös vammaisten.
- SEO (Search Engine Optimization): Prosessi, jolla parannetaan verkkosivuston liikenteen laatua ja määrää orgaanisista hakutuloksista.
- Verkkosuorituskyky: Verkkosivuston nopeuden ja skaalautuvuuden optimointi sujuvan käyttäjäkokemuksen varmistamiseksi.
- Verkkoturvallisuus: Verkkosivustojen suojaaminen hyökkäyksiltä, kuten SQL-injektio, Cross-Site Scripting (XSS) jne.
- Verkkoisännointi: Asiakkaat verkossa. Se voi olla fyysinen sijainti tai datakeskus.
- Verkkotyökalut: Työkaluja, kuten Visual Studio Code, GitHub, Netlify jne., käytetään verkkosovellusten kehitykseen.
- Kehykset: Nämä ovat valmiiksi kirjoitettua koodia, jota voidaan käyttää verkkosivujen rakentamiseen. Esimerkkejä ovat ReactJS, AngularJS ja VueJS.
- Kirjastot: Nämä ovat uudelleenkäytettäviä koodinpätkiä, joita voidaan käyttää lisäämään tiettyjä toiminnallisuuksia verkkosivustolle. Esimerkkejä ovat jQuery ja Lodash.
- Tietokannat: Näitä käytetään tiedon tallentamiseen ja hakemiseen. Esimerkkejä ovat MySQL, PostgreSQL, MongoDB.
- Backend-kehitys: Tämä sisältää palvelinpuolen ohjelmoinnin, johon kuuluu pyyntöjen ja vastausten käsittely, tietokantojen hallinta ja verkkosivuston turvallisuuden varmistaminen.
- Frontend-kehitys: Tämä sisältää asiakaspuolen ohjelmoinnin, johon kuuluu käyttöliittymän ja käyttäjäkokemuksen luominen.
- Testaus: Tämä sisältää yksikkötestauksen, integraatiotestauksen ja päästä päähän -testauksen.
- Käyttöönotto: Tämä on prosessi, jolla verkkosivusto julkaistaan internetissä.
- Ylläpito: Tämä sisältää verkkosivuston päivittämisen, virheiden korjaamisen ja turvallisuuden varmistamisen.
