### Koodilohkot

Käytä koodilohkoja esitelläksesi monirivisiä koodirimpsuja komentokorostuksilla höystettynä. Koodilohkot ovat oleellisia selkeiden koodinpätkien esittelyssä.

```mdc
```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```
```
::

Luodessasi koodilohkoa voit lisätä tiedostonimen, joka näkyy komponentin yläosassa. Ikoni ilmestyy automaattisesti, riippuen päätteestä.
Tiedostonimet helpottavat hahmottamaan koodin sijainnin ja tarkoituksen projektissa.


```md
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```
````


::tip
Jotkut kuvakkeet ovat oletuksena nimetty, mutta voit lisätä muita tiedoon `app.config.ts` avaimella `ui.prose.codeIcon`.

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    prose: {
      codeIcon: {
        terminal: 'i-ph-terminal-window-duotone'
      }
    }
  }
})
```
::

Jokaisessa koodilohkossa on mukana kopiointinappi, jolla koodin saa leikepöydälle kätevästi.

::tip
Voit muuttaa kuvaketta tiedostossa `app.config.ts` käyttäen avaimia `ui.icons.copy` ja `ui.icons.copyCheck`.

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    icons: {
      copy: 'i-lucide-copy',
      copyCheck: 'i-lucide-copy-check'
    }
  }
})
```
::

#### Korostuslinja

Korostaaksesi tiettyjä rivejä lisää `{}` haluamiesi linjanumeroiden ympärille.
Korostuslinja on kätevä tärkeiden kohtien esittelemisessä käyttäjille.

```md
```ts [nuxt.config.ts]{4-5}
export default defineAppConfig({
  ui: {
    icons: {
      copy: 'i-lucide-copy',
      copyCheck: 'i-lucide-copy-check'
    }
  }
})
```
```

## Edistyneempää

### Koodiryhmä

Koodiryhmät välilehtineen lisätään käyttämällä `code-group`. `code-group` on oiva koodin esittelyyn useilla eri kielillä tai paketinhallintaohjelmistoilla.

```md
:::code-group

```bash [pnpm]
pnpm add @nuxt/ui
```

```bash [yarn]
yarn add @nuxt/ui
```

```bash [npm]
npm install @nuxt/ui
```

```bash [bun]
bun add @nuxt/ui
```

::
```


::note{to="#pre"}
`ProsePre`-komponentin `CodeGroup` huolehtii tiedostonimistä, kuvakkeista ja kopiointinapista.
::
