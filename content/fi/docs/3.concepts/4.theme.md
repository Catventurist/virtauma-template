Virtauma perustuu Nuxt UI:n ja ottaa hyödyt irti Tailwind CSS v4:n CSS-muuttujista. Tailwind Variants API tarjoaa venyvän teemajärjestelmän.

::tip{to="https://ui.nuxt.com/getting-started/theme"}
Koko Nuxt UI-teeman kokonaiskuvan hahmottamiseksi voi lukea dokumentaatiota.
::

## Ohita käyttämällä `@theme`

Voit muokata teemaa CSS-muuttujilla [`@theme`](https://tailwindcss.com/docs/functions-and-directives#theme-directive)-ohjeilla määrittääksesi ohjelman omat suunnittelumerkit, kuten kirjaisimet, kuvat, värit, sekä reunukset.

Ne voidaan ohittaa luomalla `main.css`-tiedosto.

Näin teeman asetukset ohitetaan:

::code-group
```css [assets/css/main.css]
@import "tailwindcss";
@import "@nuxt/ui";

@theme static {
  --font-sans: 'Finlandica', sans-serif;

  --breakpoint-3xl: 1920px;

  --color-green-50: #EFFDF5;
  --color-green-100: #D9FBE8;
  --color-green-200: #B3F5D1;
  --color-green-300: #75EDAE;
  --color-green-400: #00DC82;
  --color-green-500: #00C16A;
  --color-green-600: #00A155;
  --color-green-700: #007F45;
  --color-green-800: #016538;
  --color-green-900: #0A5331;
  --color-green-950: #052E16;
}
```

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui'],
  css: ['~/assets/css/main.css']
})
```
::

## Värit

Käytetään esiasetettuja värimuuttujia, joita käytetään komponenttien tyylittelyyn sekä ulkoasun `color`-ominaiusuuksissa.

Jokainen kohta vastaa oletusta:

- :u-badge{label="primary" variant="outline"} → Pääväriä käytetään oletusvärinä komponenteissa :br [(oletus: green)]{.text-xs.text-muted}
- :u-badge{color="secondary" label="secondary" variant="outline"} → Toissijainen väri korostaa pääväriä :br [(oletus: blue)]{.text-xs.text-muted}
- :u-badge{color="success" label="success" variant="outline"} → Onnistuneisiin tiloihin :br [(oletus: green)]{.text-xs.text-muted}
- :u-badge{color="info" label="info" variant="outline"} → Tiedotuksen tiloihin :br [(oletus: blue)]{.text-xs.text-muted}
- :u-badge{color="warning" label="warning" variant="outline"} → Varoitusten tiloihin :br [(oletus: yellow)]{.text-xs.text-muted}
- :u-badge{color="error" label="error" variant="outline"} → Vihetapausten yhteydessä :br [(oletus: red)]{.text-xs.text-muted}
- :u-badge{color="neutral" label="neutral" variant="outline"} → Neutraali väri taustoille, tekstille jne. :br [(oletus: slate)]{.text-xs.text-muted}

Voit muokata värejä yleisesti päivittämällä `app.config.ts`-tiedoston `ui.colors`-avainta:

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    colors: {
      primary: 'blue',
      neutral: 'zinc'
    }
  }
})
```

## Komponentit

Värien lisäksi kaikki [Nuxt UI-komponentit](https://ui.nuxt.com/components) voidaan määritellä yleisesti tiedostossa `app.config.ts`.

Minkä tahansa komponentin ulkoasua voidaan muuttaa käyttäen komponenttien sisäisten teemaobjetien rakenteen tapaan (näkyy [komponenttisivujen lopussa](https://ui.nuxt.com/components/card#theme)).

Kaikkien painikkeiden kirjaisimien paino esimerkkinä:

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    button: {
      slots: {
        base: 'font-bold'
      }
    }
  }
})
```

Tässä `font-bold`-luokka ohittaa oletuksen `font-medium`-luokan kaikissa painikkeissa.

::note{to="https://ui.nuxt.com/components/button#theme"}
Tutustu kaikkiin mahdollisiin teeman asetuksiin komponenteissa siirtymällä **Theme**-osioon Nuxt UI-dokumentaatiosivulla.
::
