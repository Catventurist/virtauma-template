Virtauma mahdollistaa kaiken **Markdown**-sisällön luomisen sekä **komponenttien** sisällyttämisen 
käyttäen [MDC-syntaksia](https://content.nuxt.com/docs/files/markdown#mdc-syntax) pohjautuen **Nuxt Content**:n.

## Etusivu
Etusivu löytyy `index.md`-tiedostona. `MDC`-syntaksi parantaa Markdownia antaen mahdollisuuden käyttää Vue-komponentteja osioineen ja ominaisuuksineen `.md`-tiedostoissa.

::tip{to="/fi/docs/essentials/components"}
Katso lista Virtauma-proosakomponenteista, joita voi lisätä Markdowniin.
::

### Komponentit

MDC tarjoaa oman syntaksinsa, joiden avulla on helppoa käyttää Vue-komponentteja sisällössä:

```md
::u-page-feature
  #title
  Nuxt 4
::
```

### Osiot

Osioissa voi olla tekstiä tai muita komponentteja.

- **Oletusosio** tulee suoraan komponentin sisältä tai `#default`-osiosta.
- **Nimetyt osiot** saadaan käyttämällä `#`-symbolia ennen osion nimeä.

```md
::u-page-feature
  #title
  Nuxt 4
  
  #description
  Nuxt 4 tarjoaa optimoidun suorituskyvyn ja SEO:n.
::
```

### Ominaisuudet

Ominaisuudet määritellään rivinsisäisellä syntaksilla tai YAML-muodossa (frontmatter) komponentissa:

```md
  :::u-page-feature{icon="i-simple-icons-nuxt" to="https://nuxt.com"}
    #title
    Nuxt 4
    
    #description
    Nuxt 4 tarjoaa optimoidun suorituskyvyn ja SEO:n.
  :::
```

Yaml: 

```md
  :::u-page-feature
    ---
    icon: i-simple-icons-nuxt
    to: https://nuxt.com
    ---
    #title
    Nuxt 4
    
    #description
    Nuxt 4 tarjoaa optimoidun suorituskyvyn ja SEO:n.
  :::
```


::note{to="https://content.nuxt.com"}
Lue lisää MDC-syntaksista Nuxt Content-dokumentaatiossa.
::

## Dokumentaation sivut

::tip
Sisältötiedostojen ja sivujen välillä on yksi yhteen-suhde. Jokainen Markdown-sivu `content/`-kansiossa kartoitetaan suoraan sivun reiteiksi.
::

Aloitteeksesi muokkaa `.md`-tiedostoja [`content/`](https://content.nuxt.com/usage/content-directory)-kansoissa, jotta sivut päivittyvät. Reittys, navigaatio ja tekstihaku hoituu automaattisesti.

### Frontmatter

Jokainen `content/`-kansion tiedosto alkaa `---`-syntaksilla sivun alussa. Se vastaa tiedoston frontmatter:a, joka on Markdown-pohjaisten sisällönhallintaohjelmistojen yleinen tapa tarjota sivujen metadataa.

```md
  ---
  title: 'Muokkaaminen'
  description: 'Opi kirjoittamaan sisältöä.'
  ---
  
  Sisältö Markdownissa
```

### Parametrit

`/content`-kansion sivut on määritelty [page](https://content.nuxt.com/docs/collections/types#page-type)-tyyppinä Nuxt Contentissa. Ne seuraavat samoja frontmatter:n avainten rakenteita:

|               |                                          |                                                                                                             |
| ------------- | ---------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| Avain          | Tyyppi                                  | Kuvaus                                                                                                      |
| `title`       | `string`                                 | Sivun nimi. Näytetään sivun yläosassa. Käytetään `seo`-avaimena nimi, jos ei asetettuna.                    |
| `description` | `string`                                 | Sivun kuvaus. Näytetään otsiokon alla sivun yläosassa. Käytetään `seo`-avaimena kuvaus, jos ei asetettuna.  |
| `navigation`  | `boolean`                                | Jos sivu on listattuna vasemman sivupalkin navigaatiossa.                                                   |
| `layout`      | `string`                                 | Muuta Nuxt-sivun asettelua, oletuksena to `docs`.                                                           |
| `seo`         | `title: string, description: string`     | Sivuston SEO-metatiedot.                                                                                    |
